# Image Processing Web Application

## Netlify Deployment URL
https://imageprocessingwebapp.netlify.app/

## Project Description
Our application allows the user to upload an image, apply a variety of filters to it, and then download their newly edited image. The user can view both their original image and their edited image side by side to easily view changes. Our goal is to make this a useful tool for those looking to improve the look of their images in a quick, easy, and understandable way.

## Features
- Saturation Adjustment
- Gaussian Blur
- Hue Rotation
- Sepia Tone
- Emboss Effect
- Posterize Effect
- Image Upload and Download

## Technologies and Dependencies
Programming Languages: JavaScript, HTML, CSS
Frameworks and Libraries: React, Vite
Additional Dependencies: ESLint 

## Setup Instructions

### Required Installations
- Node.js (version 16.0 or higher)
- npm (comes with Node.js)
- Git

### Clone the project
1. Open a terminal on your machine
2. Navigate to the file location where you would like to store the project
3. Run this command: git clone https://github.com/TylerChasse/ImageProcessingApp.git
4. Navigate to the newly created project folder (ImageProcessingApp)

### Install Dependencies
1. Run this command: npm install

### Run Locally
1. Run this command: npm run dev
2. Navigate to the localhost url provided in a web browser
